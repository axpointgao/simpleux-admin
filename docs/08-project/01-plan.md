# 项目计划

## 一、项目概述

### 1.1 项目目标

**核心目标**：
- 构建一套完整的项目管理和业绩管理系统
- 实现商业项目、售前支持、专业贡献等业务模块
- 提供任务和工时管理功能
- 集成审批流程和系统设置

### 1.2 项目范围

**功能范围**：
- 商业项目管理
- 业绩管理
- 售前支持管理
- 专业贡献管理
- 任务和工时管理
- 审批管理
- 系统设置

**技术范围**：
- Next.js 14 (App Router)
- Supabase (PostgreSQL)
- TypeScript
- Tailwind CSS + Shadcn/UI
